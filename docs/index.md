# Quick Command Blocks

Quick Command Blocks adds one command, `/qcb`, that turns a single line of script
into a ready-made chain of pre-filled command blocks. No placing blocks by hand, no
setting repeat or chain modes, no typing into each one.

```
/qcb dir=forward; r: time set day; c: weather clear
```

## Documentation

- **[Scripting reference](SCRIPTING.md)** covers the full script format: headers,
  directions, block kinds, flags, shorthand, and worked examples.
- **[Using an AI to write scripts](AI_PROMPT.md)** is a prompt you can paste into
  ChatGPT or Claude so it writes `/qcb` scripts from a plain-language request.
